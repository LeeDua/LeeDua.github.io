# 关于Blog

博客基于VuePress实现，在default blog theme基础上参考了journey.ad和Makito的Hexo theme进行了修改。 建立博客的契机是因为看到journey.ad的blog中的看板娘心动不已，于是“给我也整一个.jpg”（果然纸片人才是第一生产力）

经过简单的了解，选择VuePress的原因如下

  • 齐名的Vue static site generator有Nuxt VuePress GridSome
    • Nuxt功能最强大，理论上Nuxt能实现VuePress和GridSome的所有功能，但也因为自由度高上手难度较大,用于blog有一种杀鸡牛刀的感觉
    • GridSome的优势在于GraphQL和外接CMS，做一个简单的blog都用不太上
    • VuePress 1.x后官方提供的blog插件对blogging涉及的逻辑有恰到好处的封装，包括markdown content-extraction、frontmatter grouping、routing等博客核心逻辑，官方的default theme完成度也很高。于是选择VuePress。
  • VuePress支持md中内嵌Vue code，将来自定义更多layout时数据耦合会十分方便。

搭建过程中的感受：

  • 前期由于VuePress的文档写的有点“零碎”，对框架的工作流没有很理解
  • 经过一番摸索并结合70-lines-theme-demo后基本弄明白了工作流
  • 而后便愉快的开始修改default theme了
  • 总体来说还是比较顺利的，从开始读文档到搭好一共三天时间,推荐尝试vuepress
